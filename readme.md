# AI Video Meeting Assistant

An AI-powered meeting assistant that can process YouTube videos or uploaded audio files, transcribe speech, summarize meetings, extract action items and decisions, and answer questions using Retrieval-Augmented Generation (RAG).

## Features

- YouTube video support
- Audio upload
- Whisper transcription
- Hindi to English translation
- AI meeting summaries
- Action item extraction
- Key decision extraction
- Question answering with RAG
- Streamlit UI

## Tech Stack

- Python
- Streamlit
- Whisper
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Mistral AI
